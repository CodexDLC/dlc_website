# Документация Дизайн-системы

В этом разделе содержатся подробные гайдлайны по дизайну, спецификации UI/UX, дизайн-токены и логика структуры сайта.

*   **`identity.md`**: Основные дизайн-токены (Цвета, Типографика, Отступы).
*   **`layout_navigation.md`**: Система сетки, брейкпоинты и структуры шапки/подвала.
*   **`ui_components.md`**: Описание состояний для кнопок, инпутов и интерактивных карточек.
*   **`page_flow.md`**: Архитектура контента для Главной страницы и направлений бизнеса.
*   **`content_map.md`**: SEO-правила, требования к ассетам и локализация.
