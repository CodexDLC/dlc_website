# 🎨 Айдентика и Стиль (Корпоративный дизайн)

## 1. Цветовая система
Цветовая палитра отражает надежность, инженерную точность и современные энергетические решения.

### Основные цвета
*   **Navy-900 (Industrial Navy):** `#0A2540` — Основной фон для навигации, футера и крупных акцентных блоков.
*   **Navy-700:** `#1A3B5C` — Состояния при наведении (hover) и второстепенные фоны.
*   **Navy-100:** `#E6EAF0` — Мягкие фоны для технических карточек.

### Акцентные цвета
*   **Green-500 (Volt Green):** `#00D084` — Основные кнопки (CTA), индикаторы прогресса, выделения.
*   **Green-600:** `#00A86B` — Состояния при наведении для кнопок.

### Нейтральные цвета / Оттенки серого
*   **Gray-900:** `#111827` — Основной цвет текста (Заголовки и основной текст).
*   **Gray-500:** `#6B7280` — Второстепенный текст, метки (labels) и вспомогательный текст.
*   **Gray-100 (Steel Grey):** `#F3F4F6` — Фоны страниц и разделяющие секции.
*   **White:** `#FFFFFF` — Фоны карточек и текст на темном фоне.

## 2. Типографика
Используются шрифты Google Fonts. Приоритет отдается читабельности и структурированной, чертежной эстетике.

*   **Шрифтовые семейства:**
    *   **Заголовки:** `Space Grotesk` или `Inter` (Геометричные, строгие, технологичные).
    *   **Основной текст:** `Roboto` (Высокая читабельность для длинных текстов).

*   **Размерная сетка (Десктоп):**
    *   **H1:** 48px / межстрочный интервал 56px, Bold (700).
    *   **H2:** 36px / межстрочный интервал 44px, Semi-Bold (600).
    *   **H3:** 24px / межстрочный интервал 32px, Medium (500).
    *   **Body Large:** 18px / межстрочный интервал 28px, Regular (400).
    *   **Body Default:** 16px / межстрочный интервал 24px, Regular (400).
    *   **Label / Caption:** 14px / межстрочный интервал 20px, Medium (500), Uppercase (заглавные с увеличенным трекингом).

## 3. Система отступов (Spacing)
Все отступы (margins, paddings) строятся на базовой сетке 8px для поддержания структурной гармонии.
*   `xs`: 8px
*   `sm`: 16px
*   `md`: 24px
*   `lg`: 32px
*   `xl`: 48px
*   `2xl`: 64px
*   `3xl`: 96px (для крупных разрывов между секциями)

## 4. Визуальные элементы
*   **Скругления (Border Radius):** Минималистичные `4px` (Строгий, инженерный вид). Избегайте полностью круглых форм, кроме специфичных бейджей.
*   **Тени:** Мягкие, направленные вниз тени для разделения слоев без эффекта "парения".
    *   *Тени карточек:* `0 4px 6px -1px rgba(10, 37, 64, 0.1)`
*   **Обработка изображений:** Чистые фото, без вшитого текста. На геро-изображениях (главных баннерах) используется темный градиент-оверлей (`rgba(10,37,64, 0.6)`) для обеспечения читабельности белого текста.
