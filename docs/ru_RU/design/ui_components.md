# 🧩 UI Компоненты и Взаимодействие

## 1. Кнопки (Инженерный стиль)
Кнопки должны ощущаться плотными и точными, с четким изменением состояний.

*   **Primary CTA (Главное действие):**
    *   *По умолчанию:* Фон `Green-500 (#00D084)`, Текст `Navy-900`, скругление `4px`.
    *   *Наведение (Hover):* Фон `Green-600 (#00A86B)`, небольшое приподнятие (Тень `sm`).
    *   *Нажатие (Active):* Фон `Green-600`, тень пропадает, масштаб `0.98`.
    *   *Неактивно (Disabled):* Фон `Gray-100`, Текст `Gray-500`, курсор `not-allowed`.
*   **Второстепенная кнопка (Secondary):**
    *   *По умолчанию:* Прозрачный фон, Обводка `2px solid Navy-900`, Текст `Navy-900`.
    *   *Наведение:* Фон `Navy-900`, Текст `White`.

## 2. Формы и Поля ввода (Inputs)
Поля ввода должны давать мгновенную валидацию для обеспечения высокого качества лидов.

*   **Поле по умолчанию:**
    *   Обводка `1px solid Gray-500`, Фон `White`, скругление `4px`.
*   **Фокус (Focus):**
    *   Обводка `2px solid Green-500`, Свечение `0 0 0 3px rgba(0, 208, 132, 0.2)`.
*   **Ошибка (Error):**
    *   Обводка `2px solid Red-500`, сообщение об ошибке под полем шрифтом `14px Red-500`.

## 3. Карточки Проектов (Referenzen)
Используются для демонстрации завершенных солнечных установок.

*   **Структура:** Изображение сверху, Контент снизу.
*   **Эффект наведения (Hover):** 
    *   Изображение слегка затемняется (оверлей `rgba(0,0,0,0.2)`).
    *   Плавное появление технических данных (например, "Мощность: 500 kWp | Срок: 3 недели").
*   **Тень:** Изначально `sm`, при наведении увеличивается до `lg` для индикации кликабельности.

## 4. Многошаговая Лид-форма (Квиз)
Двигатель конверсии сайта.
*   **Прогресс-бар:** Фиксирован сверху, заливка `Green-500` показывает процент завершения.
*   **Карточки выбора (Вместо Radio-кнопок):** Крупные кликабельные карточки (например, "Частный дом", "Завод") с SVG иконками.
    *   *Выбранное состояние:* Обводка `2px solid Green-500`, легкий фон `Green-100`.
