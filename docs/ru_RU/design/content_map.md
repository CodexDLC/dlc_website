# 🗺️ Контент-стратегия и SEO

## 1. Стратегия локализации
Сайт обслуживает две основные аудитории с различными потребностями в контенте.

*   **DE (Главный рынок):** Немецкий контент с фокусом на специфику местного рынка, государственные субсидии (KfW, BAFA), юридическое соответствие и региональное доверие (сертификаты TÜV/VDE).
*   **EN (Вторичный рынок):** Английский контент, ориентированный на корпоративный имидж, цели ESG (экология, социальная ответственность) и информацию для международных инвесторов или крупных EPC-партнеров.

## 2. Техническое SEO
Для доминирования в локальной поисковой выдаче по коммерческим и частным солнечным установкам.

*   **Микроразметка Schema.org:** Внедрение `LocalBusiness` и `EnergyBusiness` через JSON-LD для отображения точных рейтингов, местоположения и услуг в расширенных сниппетах (rich snippets).
*   **Геотаргетинг:** Заголовки H1 и H2 должны включать ключевые слова конкретного региона или города (например, "Фотоэлектрические системы для бизнеса в Берлине").
*   **Производительность:** Целевые показатели Core Web Vitals должны достигать >90 баллов на мобильных устройствах.

## 3. Медиа и Ассеты
*   **Векторная графика (SVG):** Все иконки, графики и расчеты ROI должны быть в формате SVG или сверстаны на HTML/CSS. Это обеспечивает четкость на любых экранах и позволяет легко переводить текст без редактирования самих изображений.
*   **WebP Изображения:** Все фотографии должны быть сжаты в формат WebP для сохранения высокого качества при минимальном времени загрузки. Максимальное разрешение для главных баннеров (hero) — 1920x1080.
