# 🎨 Identity & Style (Corporate Design)

## 1. Color System
The color palette represents reliability, engineering precision, and modern energy solutions.

### Primary Colors
*   **Navy-900 (Industrial Navy):** `#0A2540` — Main background for navigation, footers, and large accent blocks.
*   **Navy-700:** `#1A3B5C` — Hover states and secondary backgrounds.
*   **Navy-100:** `#E6EAF0` — Soft backgrounds for technical cards.

### Accent Colors
*   **Green-500 (Volt Green):** `#00D084` — Primary CTAs, progress indicators, highlights.
*   **Green-600:** `#00A86B` — Hover states for CTAs.

### Grayscale / Neutrals
*   **Gray-900:** `#111827` — Primary text color (Headings & Body).
*   **Gray-500:** `#6B7280` — Secondary text, labels, and helper text.
*   **Gray-100 (Steel Grey):** `#F3F4F6` — Page backgrounds and dividing sections.
*   **White:** `#FFFFFF` — Card backgrounds and text on dark backgrounds.

## 2. Typography Scale
Using Google Fonts. We prioritize readability and a structured, blueprint-like aesthetic.

*   **Font Families:**
    *   **Headings:** `Space Grotesk` or `Inter` (Geometric, strict, tech-focused).
    *   **Body Text:** `Roboto` (High legibility for long reading).

*   **Scale (Desktop):**
    *   **H1:** 48px / 56px line-height, Bold (700).
    *   **H2:** 36px / 44px line-height, Semi-Bold (600).
    *   **H3:** 24px / 32px line-height, Medium (500).
    *   **Body Large:** 18px / 28px line-height, Regular (400).
    *   **Body Default:** 16px / 24px line-height, Regular (400).
    *   **Label / Caption:** 14px / 20px line-height, Medium (500), Uppercase tracking.

## 3. Spacing System
All spacing (margins, paddings) follows a strict 8px baseline grid to maintain structural harmony.
*   `xs`: 8px
*   `sm`: 16px
*   `md`: 24px
*   `lg`: 32px
*   `xl`: 48px
*   `2xl`: 64px
*   `3xl`: 96px (used for major section gaps)

## 4. Visual Language Elements
*   **Corners / Border Radius:** Minimalist `4px` (Sharp, engineered feel). Avoid fully rounded/pill shapes except for specific badges.
*   **Shadows:** Soft, vertical-only shadows to separate layers without looking "floating".
    *   *Card Shadow:* `0 4px 6px -1px rgba(10, 37, 64, 0.1)`
*   **Image Treatment:** Clean, no embedded text. Overlay dark gradients (`rgba(10,37,64, 0.6)`) on hero images to ensure white text readability.
